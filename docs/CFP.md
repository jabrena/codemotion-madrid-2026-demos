# CFP

Description
Durante casi tres décadas, el software empresarial Java se ha implementado manualmente por ingenieros de software. En 2023, una nueva generación de herramientas con funcionalidades de IA irrumpió en el mercado, y recientemente **Cursor AI** ha captado la atención de la comunidad global de desarrolladores.

Este taller está diseñado para todos los niveles y abarca las principales funcionalidades del producto.

- Tab Model
- Modos (Ask, Plan, Agent)
- Revisión de los modelos frontera actuales
- Cursor rules para Java
- Servidores MCP
- Interfaz multiagente
- Orquestación de Cloud Agents para tareas Java

La charla explica los conceptos principales de Cursor mediante ejemplos basados ​​en Spring Boot o Quarkus.
